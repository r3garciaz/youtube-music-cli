# Keyboard Shortcuts

Complete reference of all keyboard shortcuts in youtube-music-cli.

## Global Shortcuts

These work from any screen.

| Key      | Action               |
| -------- | -------------------- |
| `?`      | Show help screen     |
| `/`      | Open search          |
| `p`      | Open plugins manager |
| `g`      | Show suggestions     |
| `,`      | Open settings        |
| `Ctrl+L` | Refresh/clear screen |
| `q`      | Quit application     |
| `Esc`    | Go back / Close menu |

## Playback Controls

| Key               | Action                              |
| ----------------- | ----------------------------------- |
| `Space`           | Play / Pause                        |
| `n`               | Next track                          |
| `→` (Right arrow) | Next track                          |
| `b`               | Previous track                      |
| `←` (Left arrow)  | Previous track                      |
| `Shift+→`         | Seek forward 10 seconds             |
| `Shift+←`         | Seek backward 10 seconds            |
| `=`               | Volume up (+5%)                     |
| `-`               | Volume down (-5%)                   |
| `s`               | Toggle shuffle                      |
| `r`               | Cycle repeat mode (off → all → one) |

## Navigation

| Key         | Action                      |
| ----------- | --------------------------- |
| `↑` / `k`   | Move selection up           |
| `↓` / `j`   | Move selection down         |
| `Enter`     | Select / Confirm            |
| `Esc`       | Go back                     |
| `Tab`       | Next result (in search)     |
| `Shift+Tab` | Previous result (in search) |

## Search

| Key     | Action               |
| ------- | -------------------- |
| `/`     | Focus search input   |
| `Enter` | Execute search       |
| `Esc`   | Clear search / Close |
| `]`     | Show more results    |
| `[`     | Show fewer results   |

## Playlist

| Key | Action               |
| --- | -------------------- |
| `a` | Add to playlist      |
| `d` | Remove from playlist |
| `c` | Create new playlist  |
| `D` | Delete playlist      |

## Plugins Manager

| Key   | Action                             |
| ----- | ---------------------------------- |
| `i`   | Install plugin                     |
| `e`   | Enable/disable selected            |
| `r`   | Remove plugin                      |
| `u`   | Update plugin                      |
| `Tab` | Switch between installed/available |

## Settings Menu

| Key       | Action           |
| --------- | ---------------- |
| `↑` / `↓` | Navigate options |
| `Enter`   | Select option    |
| `Esc`     | Close settings   |

## Help Screen

| Key     | Action     |
| ------- | ---------- |
| Any key | Close help |
| `Esc`   | Close help |
| `q`     | Close help |

## Tips

### Vim-style Navigation

If you're familiar with vim, you can use:

- `j` instead of `↓`
- `k` instead of `↑`

### Quick Access

- Press `/` anywhere to quickly search
- Press `Space` anywhere to play/pause
- Press `?` to see context-specific help

### Volume Shortcuts

Note that volume uses `=` (equals) for up and `-` (minus) for down. This avoids issues with `+` requiring Shift on most keyboards.
